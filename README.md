# Bagzzz - Handcrafted Bag Store

A clean, modern, and fully responsive e-commerce landing page template for a boutique bag brand. This project showcases a curated collection of handcrafted bags with a focus on elegant UI and smooth user experience.

## ✨ Features

*   **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
*   **Hero Section:** High-impact visual banner with a call to action.
*   **Product Discovery:** 
    *   "Recommended for You" section based on shopping habits.
    *   "Shop by Brand" slider featuring iconic logos.
*   **Interactive UI:**
    *   Search bar and language switcher.
    *   Product sorting dropdown.
    *   Feedback and Newsletter subscription forms.
*   **Product Grid:** A clean layout displaying featured products with hover effects.
*   **Detail Pages:** Individual product pages for deeper engagement.

## 🛠️ Technologies Used

*   **HTML5:** Semantic structure for better SEO and accessibility.
*   **CSS3:** Custom styling using Flexbox and CSS Grid.
*   **Google Fonts:** Modern typography for a premium feel.
